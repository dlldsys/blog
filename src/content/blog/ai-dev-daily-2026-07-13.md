## 今日要闻

### 1. OpenAI 发布 GPT-5.6 系列模型，多智能体能力正式产品化

7 月 9 日，OpenAI 正式推出 GPT-5.6 全系列模型，包含旗舰版 Sol、均衡版 Terra、轻量版 Luna 三个档位。Sol 引入全新 Ultra 模式，可默认协调 4 个 AI 子代理并行处理代码、网络安全与科研任务。API 价格较上一代最高下调 40%，Terra 与 Luna 定价分别为 $2.50/$15 和 $1/$6 每百万 Token。同日 OpenAI 宣布将于 8 月 9 日关停运营不足一年的 Atlas 浏览器，智能浏览能力将整合进 ChatGPT 桌面应用与 Chrome 扩展。

来源：[新华网](http://www.xinhuanet.com/20260710/74a4edded29942f8b0f02ac483a7a76f/c.html) · [OpenAI 官网](https://openai.com/index/gpt-5-6/) · [CSDN](https://blog.csdn.net/haohaizi_liu/article/details/162759916)

### 2. Anthropic 发布 Claude Sonnet 5，主打低成本 Agent 循环

Anthropic 发布 Claude Sonnet 5，性能接近 Opus 4.8， introductory 定价 $2/$10 每百万输入/输出 Token（8 月 31 日后恢复至 $3/$15），专为低成本 Agent 循环设计。同时 Anthropic 将 Fable 5 访问权限延长至 7 月 19 日，Claude Code 每周限额提升 50%。需要注意的是 Sonnet 5 采用新分词器，相同文本可能产生多达 1.35 倍的 Token，实际成本需以真实流量为准。

来源：[Anthropic 官网](https://www.anthropic.com/news/claude-sonnet-5) · [TechCrunch](https://techcrunch.com/2026/06/30/anthropic-launches-claude-sonnet-5-as-a-cheaper-way-to-run-agents/) · [dreaming.press](https://dreaming.press/posts/2026-07-10-model-shuffle-gpt56-sonnet5-gemini35-for-founders.html)

### 3. Google 发布 Gemini 3.5 Flash，旗舰版 Pro 延期

Google 按时发布 Gemini 3.5 Flash，定价 $1.50/$9 每百万 Token，支持 100 万 Token 上下文窗口，缓存输入低至 $0.15。旗舰版 Gemini 3.5 Pro 因企业测试中发现推理与编码能力回退而延期至 7 月。Google 同时推出 Antigravity 2.0 智能体栈，包括 Antigravity CLI、SDK 以及 Gemini API 中的托管智能体功能。

来源：[dreaming.press](https://dreaming.press/posts/2026-07-10-model-shuffle-gpt56-sonnet5-gemini35-for-founders.html)

### 4. SpaceXAI 联手 Cursor 发布 Grok 4.5，对标 Claude Opus 4.8

7 月 9 日，SpaceXAI 发布新一代大模型 Grok 4.5，这是其与 AI 编程公司 Cursor 联合训练的首款专攻代码与智能体任务的模型，依托数万台 NVIDIA GB300 GPU 完成训练。官方数据显示 Grok 4.5 在关键基准测试中全面超越 Claude Opus 4.8，运行速度达 80 TPS，Token 利用效率为竞品两倍，单次任务步骤减半。定价 $2/$6 每百万 Token，不到同类模型一半。

来源：[CSDN](https://blog.csdn.net/haohaizi_liu/article/details/162759916)

### 5. 智谱 AI 完成 314 亿港元融资，发布"摸高计划"两年战略

智谱 AI 宣布以每股 1588 港元配售最多 1978 万股新 H 股，预计募资约 314.11 亿港元，创下港股 AI 企业单笔最大融资纪录。同日创始人唐杰发布"TouchHigh（摸高）"两年战略规划，明确未来两年不追求短期商业变现，集中资源攻坚长程任务能力、自治智能体系统、完全自我训练、安全治理四大核心技术方向。

来源：[CSDN](https://blog.csdn.net/yuntongliangda/article/details/162807778)

## 新工具 & CLI

### 1. OpenAI openai-cli — 终端中的 AI 全能助手

OpenAI 发布开源命令行工具 openai-cli（Apache 2.0 许可），支持通过 Homebrew 或 Go 安装。可直接在终端调用 OpenAI 全套能力，包括图像生成、音频转录、Web 搜索、代码解释、文件检索等。输出支持 JSON/YAML 格式，内置 GJSON 语法可像 jq 一样快速提取字段，完美适配 Unix 管道工作流。

来源：[AI Damn](https://ai-damn.com/openai-s-new-cli-tool-brings-ai-power-to-your-terminal-1778213693127)

### 2. Augment Auggie CLI — 理解整个代码仓库的终端代理

Augment 发布 Auggie CLI 工具，作为代理型 CLI 拥有强大的上下文引擎，可理解整个代码仓库。支持与 VS Code 原生集成，专为无缝融入现有开发工作流而设计，定位企业级 AI 开发平台。

来源：[AI Damn](https://ai-damn.com/augment-s-auggie-cli-aims-to-transform-developer-workflows-1754263237246)

### 3. Cursor 3.11 — Side Chats 并行对话与对话搜索

Cursor 发布 3.11 版本，核心新功能 Side Chats 允许在主线程旁开启并行对话（`/side` 或 `/btw`），继承上下文且不打断主流程，可通过 @-mention 将结论召回。对话搜索（Cmd+K）可跨数千条对话本地索引查找历史线程。项目和仓库选择器也经过重新设计。

来源：[Cursor Changelog](https://cursor.com/changelog) · [jls42.org](https://jls42.org/en/news/ia-actualites-12-jul-2026)

### 4. GitHub Issue Fields 正式可用

GitHub 宣布 Issue Fields 功能正式发布（GA），允许在 Issue 上直接添加结构化类型元数据（优先级、预估工作量、日期、自定义值），无需逐一打开 Issue 即可快速查看状态，适用于管理大量工单的团队。

来源：[GitHub on X](https://x.com/github/status/2076375438104908255) · [jls42.org](https://jls42.org/en/news/ia-actualites-12-jul-2026)

### 5. HuggingFace HuggingNews — 微调开源模型超越 Claude 和 GPT

Hugging Face 发布 HuggingNews，一个微调的小型开源权重模型，号称在撰写 AI 新闻方面超越 Claude 和 GPT，展示了基于开源基础设施构建领域专用自主内容智能体的模板。

来源：[The Agent Times](https://www.theagenttimes.com/articles/hugging-face-ships-fine-tuned-open-model-that-outperforms-cl-8e6ebeb1)

## 编程方式

### 1. Amp 切换默认 Agent 模型至 GPT-5.6，成本减半

Sourcegraph 旗下 Amp 宣布将默认 Agent 模型从 Claude Opus 切换至 GPT-5.6 Sol X-High，Fable 5 High 作为"Oracle"模型负责审查验证。CEO Quinn Slack 表示平均成本降低约 50%，且响应质量更优。这标志着模型竞争已从原始基准测试转向真实 Agent 场景下的性价比比拼。

来源：[Quinn Slack on X](https://x.com/sqs/status/2075802047329484895) · [jls42.org](https://jls42.org/en/news/ia-actualites-12-jul-2026)

### 2. Stanford AgentFlow — 模块化强化学习 Agent 框架

斯坦福大学发布 AgentFlow 框架，由 Planner、Executor、Verifier、Generator 四个模块组成，通过 Flow-GRPO 训练方法将长期奖励优化转化为可管理的步骤更新。在知识密集型搜索（+14.9%）、Agent 推理（+14.0%）、数学任务（+14.5%）等领域显著超越基线，7B 模型在部分基准上甚至超越 GPT-4o。工具调用错误减少 28.4%，MIT 许可证开源。

来源：[AI Damn](https://ai-damn.com/stanford-unveils-agentflow-modular-ai-framework-1760051632561)

### 3. ActiveGraph — 以事件日志为核心的 Agent 运行时

ActiveGraph 颠覆传统 Agent 框架，将事件日志作为 Agent 行为和状态的唯一真相来源，支持确定性回放、低成本 fork 分支以及端到端血缘追踪，为自我改进型 Agent 提供增强的调试与开发能力。

来源：[Qlankr News](https://news.qlankr.com/article/the-log-is-the-agent-mr7bi316)

### 4. 模型商品化与多供应商策略成为新范式

本周 GPT-5.6、Sonnet 5、Gemini 3.5 Flash 三家中端模型同台竞技，定价战已转移至中端市场。开发者最佳实践变为：基于真实流量选择模型（关注输出 Token 量和真实 Token 计数），同时保留第二供应商接入能力——模型持续迭代，在供应商间灵活切换才是持久优势。监管风险（如 GPT-5.6 被白宫限制 12 天、Fable 5 被商务部下架）也使多供应商抽象层成为业务连续性的必需品。

来源：[dreaming.press](https://dreaming.press/posts/2026-07-10-model-shuffle-gpt56-sonnet5-gemini35-for-founders.html)

## 总结

本周 AI 行业进入"模型商品化 + 智能体产品化"双轮驱动阶段：OpenAI、Anthropic、Google 三家在一周内同时刷新中端模型定价，多智能体并行能力从概念走向产品落地，而工具链侧（openai-cli、Cursor Side Chats、AgentFlow）正让 AI 编程工作流变得更加结构化和可组合。
